## Findings Resolved During Audit

### FINDING-01: ink-400 token missing (HIGH severity, RESOLVED)

**Issue:** 15 usages of `border-ink-400` / `bg-ink-400` across 7 files (Nav, Hero, About, Skill, Experience, Contact, Footer). Token was not defined in `tailwind.config.js` — only ink 950/900/800/700/600/500 were declared. Tailwind JIT generated `border-color` with `currentColor` fallback, causing borders to inherit text color (subtle visual defect, not invisible).

**Resolution:** Added `400: '#3A3A6A'` to ink palette in tailwind.config.js (commit 5d40f15). Maintains linear darkening progression. Build + lint verified clean post-fix.

**Affected requirements:** DSGN-01, CONT-01..08, NAV-01, HERO-05, RESP-01 — all now correctly satisfied.

## Tech Debt (Non-Blocking)

### t.hero.cta2 orphaned key (Phase 2)
Hero CV buttons hardcode "Download CV (EN)" / "Descargar CV (ES)" instead of consuming `t.hero.cta2`. Functionally correct but bypasses i18n system. Future locale additions would require code change instead of translation file edit. **Action:** Optional refactor in v3.5 cleanup pass.

### t.contact.loc orphaned key (Phase 3)
Contact section has no Location card by design decision D-10 (Phase 3) — phone/LinkedIn/GitHub only. Translation key remains defined. **Action:** Remove key in v3.5 i18n cleanup.

### scripts/og-template.html uses Google Fonts CDN (Phase 3)
OG image generator script loads fonts via fonts.googleapis.com. Generator-only, no runtime impact (og-image.png is committed, never regenerated in production). Conflicts with INFRA-05 self-host contract only if regenerating in network-restricted environment. **Action:** Migrate generator to self-hosted woff2 import in v3.5 maintenance.

### GA script HTML position (Phase 3)
`<script>` for G-4TZJGR3MXR placed between `</body>` and `</html>` in index.html. HTML spec violation — should be in `<head>` or before `</body>`. Functionally fires correctly. **Action:** Move to `<head>` (synchronous) or before `</body>` in v3.5 cleanup.
