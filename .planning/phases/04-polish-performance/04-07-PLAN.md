   - Open `npm run preview` build in Chrome
   - DevTools → Performance tab → CPU 4x throttle + Network "Slow 4G"

   - Verify: FPS stays 55-60, no long tasks &gt; 50ms, no layout-thrash from useInView observers, no jank from Hero h1b char-reveal during scroll

     - 4K desktop: 2560 × 1440
   - Per viewport, scroll the entire page top-to-bottom and check:
     - No horizontal overflow (scrollbar at bottom)
     - No clipped text or images (none expected — confirmed in Plan 04-03 audit)
     - Mobile menu hamburger appears on widths &lt; 768 (md breakpoint)
     - LangPill renders inline on desktop, inside mobile menu on phones
     - Stats grid: 2 columns on phone (`grid-cols-2`), 4 columns on `sm:` and up
     - Experience timeline: vertical rail visible, dots aligned, cards readable
     - Contact email card: text doesn't wrap awkwardly at 4K (has `break-all` for fallback)
     - Footer social links: spaced consistently
   - Note any discovered issues — small fixes are acceptable in this plan; large fixes require re-engaging planner

   - Skip-link (Plan 04-04) surfaces on first Tab + Enter jumps to `#main`
