## Pointer-pick rewrite (Alert A12 / MOD-08)

Replace the existing 2D ortho pick math (`WebGLConstellation.js:~619-621`):

With perspective-aware projection:

```
import { Vector3 } from 'three'  // add to top-of-file imports
```
