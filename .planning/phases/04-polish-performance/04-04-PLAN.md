Skip-link CSS pattern (industry standard):
```css
.skip-link {
  position: absolute;
  top: -100px;
  left: 0;
  z-index: 1000;
  padding: 0.75rem 1rem;
  background: var(--color-brand);
  color: var(--color-text-inverse);
  font-family: 'JetBrains Mono', monospace;
  font-size: 0.75rem;
  font-weight: 800;
  border-radius: 0 0 0.5rem 0;
  transition: top 150ms ease-out;
}
.skip-link:focus {
  top: 0;
  outline: 2px solid var(--color-brand-light);
  outline-offset: 2px;
}
```

The `top: -100px` hides it visually; `:focus` slides it down to `top: 0`. Screen readers see it always (no `display: none`). Keyboard-only users see it on Tab.
