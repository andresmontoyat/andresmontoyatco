## Overview

Complete brownfield redesign of an existing React + Tailwind portfolio site for Carlos Andrés Montoya — Solutions Architect & Senior Backend Engineer. Replaced dated neon cyan/purple theme with a bold indigo/coral palette, modernized the build toolchain (CRA → Vite 6, React 17 → 18, Tailwind v2 → v3.4), self-hosted fonts, rewrote all 6 page sections with scroll-triggered entrance animations, added a 12-entry vertical experience timeline with per-card expand/collapse, email-hero contact card with copy-to-clipboard, branded Open Graph rich link preview, and hit Lighthouse Performance 98 / A11y 100 / Best Practices 100 / SEO 100 on mobile.
